图像恢复

Textual Prompt Guided Image Restoration

Qiuhai Yan, Kang Chen, Aiwen Jiang, Long Peng, Qiaosi Yi, Chunjie Zhang

本文提出了一种基于文本提示的图像恢复方法，通过引入语言模型的语义指导能力，有效地提升了图像恢复的质量。我们的方法能够根据文本描述中的语义信息，针对性地恢复图像中的不同区域，同时保持整体一致性。实验结果表明，我们的方法在多种图像恢复任务中取得了先进的性能。该方法不仅能够处理常见的图像恢复问题，还能根据文本描述进行特定区域的增强和优化，为图像恢复领域引入了新的控制维度。

散景渲染

Self-supervised multi-scale pyramid fusion networks for realistic bokeh effect rendering

Zhifeng Wang, Aiwen Jiang, Chunjie zhang, Hanxi LI, Bo Liu

本研究提出了一种自监督多尺度金字塔融合网络，用于实现真实的散景效果渲染。我们利用多尺度特征提取和金字塔结构融合，有效地捕捉图像中的深度信息和空间关系，从而生成自然且真实的散景效果。该方法无需真实的散景对作为训练数据，大大降低了数据采集的难度。通过自监督学习策略，网络能够从单张图像中提取各种尺度的特征信息，并通过精心设计的融合机制重建具有真实感散景效果的图像。实验结果表明，我们的方法在视觉质量和定量评估上优于现有的监督学习方法。

Vision Transformer 散景渲染

A Dense Prediction ViT Network for Single Image Bokeh Rendering

Zhifeng Wang, Aiwen Jiang

本文首次将Vision Transformer应用于单图像散景渲染任务，提出了一种密集预测ViT网络。通过结合Transformer的全局建模能力和密集预测的局部细节恢复能力，我们的方法能够生成具有逼真景深效果的高质量散景图像。实验结果表明，该方法在视觉质量和定量评估上都优于现有的卷积神经网络方法。通过全局注意力机制，我们的模型能够更好地理解场景深度关系，从而更准确地区分前景和背景，实现更自然的散景效果。此外，我们还提出了一种新的损失函数，进一步提高了渲染结果的真实感和自然度。

图像去雨 集成学习

Ensemble single image deraining network via progressive structural boosting constraints

Long Peng, Aiwen Jiang, Haoran Wei, Bo Liu, Mingwen Wang

本文提出了一种基于渐进式结构增强约束的单图像去雨集成网络。通过引入结构保持约束和多阶段渐进式学习策略，有效地解决了现有去雨方法在保持图像结构和纹理细节方面的局限性。我们的方法通过多个子网络的集成决策，在多种复杂雨天场景中都表现出优异的去雨效果和结构保持能力。实验表明，相比于单一网络结构，我们的集成方法能够更好地处理不同密度和类型的雨线，同时有效避免过度平滑和细节丢失的问题。该方法在多个基准数据集上都取得了当前最先进的性能。

图像去雾

Efficient and accurate multi-scale topological network for single image dehazing

Qiaosi Yi, Juncheng Li, Faming Fang, Aiwen Jiang, Guixu Zhang

本文提出了一种高效且准确的多尺度拓扑网络用于单图像去雾。通过引入拓扑结构感知模块和多尺度特征融合机制，该方法能够有效地处理不同类型和密度的雾霾，同时保持图像的拓扑结构和边缘信息。与现有方法相比，我们的网络在计算效率和去雾质量方面都取得了显著进步。实验表明，该方法能够在保持图像自然性的同时有效去除雾霾影响，并在多个公开数据集上取得了最先进的性能。

风格迁移 视频处理

Deliberation on object-aware video style transfer network with long–short temporal and depth-consistent constraints

Yunxin Liu, Aiwen Jiang, Jiancheng Pan, Jianming Liu, Jihua Ye

本文针对视频风格迁移中的时间一致性问题，提出了一种基于对象感知的视频风格迁移网络，结合长短时序约束和深度一致性约束。我们的方法能够识别视频中的对象，并在风格迁移过程中保持对象的一致性，同时考虑场景深度信息，生成时间上连贯且视觉上稳定的风格化视频。该方法解决了传统视频风格迁移中常见的闪烁和时间不连续问题，为视频艺术创作提供了新的可能性。

图像去雾

MSNet: A novel end‐to‐end single image dehazing network with multiple inter‐scale dense skip‐connections

Qiaosi Yi, Aiwen Jiang, Xiaolin Deng, Changhong Liu

本文提出了一种名为MSNet的端到端单图像去雾网络，其特点是采用多尺度密集跳跃连接结构。这种设计能够在不同尺度下有效融合特征，增强网络对雾霾场景中细节和结构的恢复能力。实验表明，MSNet在各种复杂雾霾场景下都能生成清晰、自然的去雾结果，并在多个基准数据集上取得了优异的性能。通过多尺度特征的有效利用，我们的方法解决了传统去雾方法在处理不同密度雾霾时的适应性问题。

图像去雨

Cumulative rain density sensing network for single image derain

Long Peng, Aiwen Jiang, Qiaosi Yi, Mingwen Wang

本文提出了一种累积雨密度感知网络用于单图像去雨。与现有方法不同，我们的网络能够自适应地估计图像中的雨密度分布，并根据雨密度信息调整去雨策略。通过引入累积雨密度感知机制，该方法能够有效地处理不同密度和类型的雨线，在保持图像背景细节的同时去除雨水干扰。实验结果表明，我们的方法在多种复杂雨天场景下都表现出优异的去雨效果，具有较强的实用性和鲁棒性。

人群分析

Static Crowd Scene Analysis via Deep Network with Multi-branch Dilated Convolution Blocks

Haoran Liu, Aiwen Jiang, Qiaosi Yi, Xiaolin Deng, Jianyi Wan, Mingwen Wang

图像超分辨率

Deep residual refining based pseudo‐multi‐frame network for effective single image super‐resolution

Kangfu Mei, Aiwen Jiang, Juncheng Li, Bo Liu, Jihua Ye, Mingwen Wang
